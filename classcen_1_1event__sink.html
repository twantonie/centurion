Detailed Description

template<typename E>
class cen::event_sink< E >

Manages a subscription to an event.

Note
This class is used by event_dispatcher and isn't really meant to be used by itself.
Template Parameters
Ethe event type, e.g. window_event.
See also
event_dispatcher
