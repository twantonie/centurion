Detailed Description

template<typename Derived>
class cen::haptic_effect< Derived >

Represents a haptic effect.

The following is an illustration of the different stages of a haptic effect, copied from the SDL documentation, albeit with tweaked terms. 

 Strength
 ^
 |
 |    effect level -->  _________________
 |                     /                 \
 |                    /                   \
 |                   /                     \
 |                  /                       \
 | attack_level --> |                        \
 |                  |                        |  <---  fade_level
 |
 +--------------------------------------------------> Time
                    [--]                 [---]
                    attack_length        fade_length

 [------------------][-----------------------]
 delay               duration

Derivedthe type of the subclass, for CRTP.

5.2.0
