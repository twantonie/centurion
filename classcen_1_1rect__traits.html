Detailed Description

template<typename T, enable_if_convertible_t< T, int, float > = 0>
class cen::rect_traits< T, >

Provides rectangle traits used by basic_rect.

Note
Whilst it is possible to supply a type that isn't int or float, rectangles will always use one of them as the representation type.
Template Parameters
Tthe representation type, must be convertible to int or float.
