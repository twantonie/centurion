◆ data_1()

auto cen::window_event::data_1 ( ) const -> i32
inlinenoexcept

Returns the value of the first data value.

The meaning of this value is dependent on the window event ID of this window event.

For instance, if the event ID is SizeChanged, then data1 and data2 represent the new width and height of the window respectively. See the WindowEventID documentation for more details about whether the value returned from this method is meaningful in regard to the window event ID.

Returns
the value of the first data value.
