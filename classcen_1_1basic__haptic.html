Detailed Description

template<typename T>
class cen::basic_haptic< T >

Represents a haptic (force feedback) device.

This class is a part of the owner/handle API, the owner/handle versions of this class are haptic/haptic_handle

See also
haptic
haptic_handle
Since
5.2.0
